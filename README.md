# SPA React-js

## 📌 Описание проекта

Приложение демонстрирует:

* авторизацию пользователя
* получение и отображение списка продуктов
* просмотр детальной информации о продукте

Все данные получаются из [DummyJSON API](https://dummyjson.com/).

---

## 🛠️ Требования

* React
* React Router
* Redux Toolkit
* RTK Query
* Webpack
* Babel
* CSS (без UI библиотек)

---

## ⚙️ Установка и запуск

### 1. Установка зависимостей

```
npm install
```

### 2. Запуск в dev режиме

```
npm start
```

### 3. Сборка проекта

```
npm run build
```
