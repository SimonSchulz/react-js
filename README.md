# SPA React Application (DummyJSON)

## 📌 Описание проекта

Одностраничное приложение (SPA), реализованное на React с использованием современного стека фронтенд-разработки.

Приложение демонстрирует:

* авторизацию пользователя
* получение и отображение списка продуктов
* просмотр детальной информации о продукте

Все данные получаются из DummyJSON API.

---

## 🚀 Функциональность

* 🔐 Авторизация (DummyJSON `/auth/login`)
* 📦 Список продуктов (с пагинацией)
* 📄 Детальная страница продукта
* 🔒 Защищённые роуты (без токена доступ закрыт)
* 💾 Сохранение токена в localStorage
* ⚡ Кэширование данных через RTK Query
* 📱 Адаптивная вёрстка

## ⚙️ Установка и запуск

### 1. Установка зависимостей

```
npm install
```

### 2. Запуск в dev режиме

```
npm start
```

Приложение будет доступно на:

```
http://localhost:3000
```

### 3. Сборка проекта

```
npm run build
```
